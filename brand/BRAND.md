# ClauseKeeper — Brand Guide

Building a mini-brand, not just a product. Keep everything flush with these tokens.

## Logo
- Shield mark: navy outline + 3 navy document lines (centered on shield axis) + teal checkmark seal (bottom-right) with white separator ring.
- Source SVGs: `brand/logo-icon.svg`, `brand/wordmark.svg`
- Exports: `brand/profile-400.png` (X profile), `brand/header-1500x500.png` (X header)
- Render pipeline: headless Chrome screenshot of SVG (no image-gen key in env).

## Wordmark
- "Clause" = navy, "Keeper" = teal. Generous gap between mark and the "C".
- Subtitle: **ALWAYS-CURRENT LEGAL DOCUMENTS** — always spell "Documents" out fully, never abbreviate.

## Colors
- Background: **white** (#FFFFFF) — bright, makes the mark pop.
- Navy: **#0B2237** (primary text, shield, "Clause")
- Teal: **#2DD4BF** → **#14B8A6** gradient (accent, "Keeper", checkmark, CTAs)
- Muted slate: **#5B7488** (subtitles), border edge **#E2E8F0**

## Voice / positioning
- Honest, trust-first: "a lawyer-quality starting point in minutes, kept current as laws change" — never "guaranteed compliance".
- Free scanner = lead magnet. Paid = the fix, kept current.

## Image-gen tools (when raster/photographic assets needed)
- Preferred: **krea.ai**, **Banana (Nano Banana Pro)**. (Env has no image-gen key wired; logos done as SVG.)
