# ClauseKeeper SEO Drafts

Review-ready Markdown drafts generated by `scripts/generate_seo_draft.py`.

## Generate the next draft

```bash
python3 scripts/generate_seo_draft.py
```

Useful options:

```bash
python3 scripts/generate_seo_draft.py --list
python3 scripts/generate_seo_draft.py --topic privacy-policy-ai-disclosure-examples
python3 scripts/generate_seo_draft.py --output-dir content/drafts
```

Each draft includes front matter, target keyword, CTA back to the free scanner, FAQ copy, and JSON-LD FAQ schema for publication review.

Before publishing: verify jurisdiction-specific claims and examples with qualified counsel. ClauseKeeper is not a law firm and does not provide legal advice.
